# SyncroProyect
Proyecto web para visualizar empleados, revisar evaluaciones y crear equipos de trabajo de forma automática. Incluye una interfaz moderna con dashboard, gestión de personal y un Team Builder que permite filtrar empleados y generar propuestas de equipos según distintos criterios.


Resumen general del proyecto
El proyecto está formado por 4 páginas HTML y un archivo JavaScript central que comparten la misma idea: mostrar información de empleados y ayudar a crear equipos de trabajo de forma visual y organizada.

1. Dashboard
Es la página de inicio o vista general. Muestra métricas y un panorama rápido del estado del equipo, como rendimiento, disponibilidad o actividad. Sirve para tener una visión general rápida.

2. Employees
Esta página está enfocada en la gestión de empleados. Muestra la información de cada persona, su rol, equipo y datos relevantes. Es la vista para consultar o revisar el personal disponible.

3. Evaluations
Aquí se presenta la información de evaluaciones o rendimiento. Se enfoca en mostrar resultados y comparaciones relacionadas con los empleados, útil para analizar desempeño.

4. Team Builder
Es la página más interactiva. Permite crear equipos, seleccionar empleados, filtrar por criterios y generar propuestas automáticas de grupos. También muestra equipos existentes y ayuda a comparar opciones.

5. Base JavaScript
El archivo JavaScript actúa como base de datos lógica del sistema. Allí se cargan los empleados, se organizan sus datos y se implementan las reglas para generar propuestas de equipos. También ayuda a mantener la información compartida entre las páginas.
