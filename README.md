# SyncroProyect
Proyecto web para crear equipos de trabajo de forma automática que incluye un filtro empleados y genera propuestas de equipos según distintos criterios.

# Usar el programa
- Exportar en .zip
- Extraer en una carpeta
- Abrir la carpeta en una hoja blanca de visual studio code
- Ejecutar el codigo
- (Es recomendable usar LiveServer que es una extensionde Visual Studio Code)

Resumen general del proyecto
para tener una visión general rápida.

1. Team Builder
Es la página más interactiva. Permite crear equipos, seleccionar empleados, filtrar por criterios y generar propuestas automáticas de grupos. También muestra equipos existentes y ayuda a comparar opciones.

2. Base JavaScript
El archivo JavaScript actúa como base de datos lógica del sistema. Allí se cargan los empleados, se organizan sus datos y se implementan las reglas para generar propuestas de equipos. También ayuda a mantener la información compartida entre las páginas.
